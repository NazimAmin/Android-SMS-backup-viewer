#### About: Takes message backup in XMl and displays as a chat view to read
#### Why? To read old messages from old android devices

```js<smses>
                  <sms
                    address="3214348976"
                    date="315967535846"
                    type="2"
                    body="Your message string"
                    readable_date="Jan 6, 1980 6:45:35 AM"
                  />
                  ...
                </smses>
```
